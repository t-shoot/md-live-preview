# アウトライン(見出し一覧) — 基本設計書

## 概要・システム構成
Markdown Live Preview Editorのアクティビティバーに追加する、見出し一覧表示ビュー。
VS Code標準の「アウトライン」パネルはWebviewベースのカスタムエディタに対応しないため(アクティブな*テキストエディタ*専用)、代わりに拡張機能自身のサイドバービューとして実装する。

## 機能一覧
| 機能 | 説明 |
|---|---|
| 見出し一覧表示 | アクティブなMarkdown Live Previewの見出し(h1〜h6)を一覧表示 |
| クリックでジャンプ | 一覧の項目をクリックすると、エディタ側で該当行にカーソル移動・スクロール |
| タブ追従 | 複数の`.md`をLive Previewで開いているとき、アクティブなタブに追従して一覧を更新 |
| 編集追従 | 見出しの追加・削除・編集内容にあわせて一覧を再生成(150msデバウンス) |
| 未オープン時の案内 | Live Previewが1つも開かれていない/アクティブでないときは案内文を表示 |

## 画面/インターフェース一覧
- アクティビティバー「Markdown Live Preview Editor」→「アウトライン」ビュー(Webview)。

## 外部連携仕様
- ホスト↔アウトラインWebview: `HostToOutlineMessage`(`update`/`noDocument`)、`OutlineToHostMessage`(`ready`/`jumpToHeading`)。
- ホスト↔エディタWebview: `HostToEditorMessage`に`jumpToLine`を追加し、アウトラインからのジャンプ要求をエディタ側へ中継する。
