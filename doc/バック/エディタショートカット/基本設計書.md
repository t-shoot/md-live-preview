# エディタショートカット — 基本設計書

## 概要・システム構成
Markdown Live Preview Editor(CodeMirror 6ベースのWebviewエディタ)に、装飾・検索用のキーボードショートカットを追加する機能群。

## 機能一覧
| 機能 | ショートカット | 説明 |
|---|---|---|
| 太字トグル | `Ctrl+B`(`Cmd+B`) | 選択範囲を `**...**` で囲む/外す |
| 斜体トグル | `Ctrl+I`(`Cmd+I`) | 選択範囲を `*...*` で囲む/外す |
| 検索パネルを開く | `Ctrl+F`(`Cmd+F`) | 検索・置換パネルを開く(検索欄にフォーカス) |
| 置換パネルを開く | `Ctrl+H`(`Cmd+H`) | 検索・置換パネルを開く(置換欄にフォーカス) |
| 次を検索 / 前を検索 | `F3`/`Ctrl+G`、`Shift+F3`/`Shift+Ctrl+G` | `@codemirror/search` 既定のバインド |
| 行へ移動 | `Ctrl+Alt+G` | `@codemirror/search` 既定のバインド |
| 次の一致を選択 | `Ctrl+D` | `@codemirror/search` 既定のバインド |

## 画面/インターフェース一覧
Markdown Live Preview のエディタ画面(Webview)内で完結する。検索・置換パネルは `@codemirror/search` が提供する標準UI(検索欄・置換欄・オプション・ボタン一式)をそのまま使用する。

## 外部連携仕様
- ホスト(拡張機能本体)との通信は不要。Webview内のCodeMirror拡張・キーマップのみで完結する。
- 依存パッケージ: `@codemirror/search`(検索・置換UI一式を提供)。
