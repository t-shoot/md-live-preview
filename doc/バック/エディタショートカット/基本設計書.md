# エディタショートカット — 基本設計書

## 概要・システム構成
Markdown Live Preview Editor(CodeMirror 6ベースのWebviewエディタ)に、装飾・検索用のキーボードショートカットを追加する機能群。

## 機能一覧
| 機能 | ショートカット | 説明 |
|---|---|---|
| 太字トグル | `Ctrl+B`(`Cmd+B`) | 選択範囲を `**...**` で囲む/外す |
| 斜体トグル | `Ctrl+I`(`Cmd+I`) | 選択範囲を `*...*` で囲む/外す |

**検索・置換(`Ctrl+F`/`Ctrl+H`)は今回のリリースから一旦取り下げ**(下記「取り下げについて」参照)。

## 画面/インターフェース一覧
Markdown Live Preview のエディタ画面(Webview)内で完結する。

## 外部連携仕様
ホスト(拡張機能本体)との通信は不要。Webview内のCodeMirror拡張・キーマップのみで完結する。

## 取り下げについて(検索・置換)
`@codemirror/search` を使った検索・置換パネル(`Ctrl+F`/`Ctrl+H`)を一度実装したが、VS Code標準のFind Widgetのような見た目への調整が難航したため、2026-07-23時点でこのリリースから一旦取り下げた。
- `src/webview-editor/main.ts` から `search()` 拡張・`searchKeymap`・`Ctrl+H`用の関数の配線を削除済み。
- 見た目調整用のCSS(`media/webview-editor-theme.css`)と依存パッケージ(`@codemirror/search`)は、再開時にすぐ使えるよう削除せず残している。
- 詳細設計書側の記述も、再開時の参考として残している。
